MSK Exam Practice (TextEdit version) - Created and maintained by Rohaan Syan, MD Candidate, University of Ottawa.

You can open it locally with any browser.

How to run (Mac):
1) Unzip the folder.
2) Double-click index.html (it will open in Safari/Chrome).
3) If images don't load, make sure the 'static' folder is beside index.html.

Files:
- index.html  (default entry; redirects to title screen)
- title.html  (title screen)
- msk.html    (MSK module home - Minimal Pro + Atlas scanability hybrid)
- exam.html   (study page; uses ?exam=shoulder etc.)
- quiz.html   (quiz page; uses ?exam=shoulder etc.)
- msk_data.js (your MSK_EXAMS data, embedded as JSON)
- static/     (images)
- assets/     (css and js)
